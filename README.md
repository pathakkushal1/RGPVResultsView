# RGPVResultsView
By using this we can view multiple student results at once on single screen in table format we can see the SGPA and CGPA of student and Remark of pass or Fail.
To use this save the RGPV result pages and give path to program of those files it will automatically read and give you a JFrame and also give you a .CVV file to create graphs of any other representaion of the data.
